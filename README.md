📖 Descripción del Proyecto
Este proyecto consiste en el desarrollo de una plataforma web estática de gestión de proyectos, construida utilizando HTML.
La aplicación simula un entorno donde los usuarios pueden visualizar información general, explorar proyectos, ver detalles específicos y consultar su perfil.
El objetivo principal es aplicar conceptos de estructuración de páginas web, organización de contenido y navegación entre distintas vistas.
